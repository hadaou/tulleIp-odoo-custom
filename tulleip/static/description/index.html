TulleIP

TulleIP is a comprehensive inventory management solution that extends the functionality of the standard Odoo product categories.

Key Features:

  • Image Support: Each product category can have an associated image, which can be loaded from a URL or a local file.
  • Category Groups: Product categories can be grouped together for easier management.
  • Product Tags: Product categories can have multiple associated tags.
  • Product Attributes: Each product category can have multiple associated attributes, each with its own set of values.
  • Public Categories: Product categories can be associated with multiple public categories.
  • Add Product Wizard: A wizard is provided for adding new products to a category, with support for setting attributes, tags, and public categories.
  • Child Count: The number of child categories for each category is automatically calculated and stored.

With TulleIP, you can easily manage your products, track inventory levels, and analyze inventory data.
